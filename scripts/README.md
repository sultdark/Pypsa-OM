# Analysis scripts

This folder is reserved for the Oman-specific scripts used to validate, analyse and visualise the PyPSA-Oman scenarios.

The public release should contain the exact scripts used to:

- calculate 2024 validation indicators against official annual electricity data;
- aggregate generation, installed capacity, direct CO2 emissions and VRE curtailment;
- calculate transmission utilization and reinforcement indicators;
- summarize battery and hydrogen storage and conversion results; and
- create the repository and publication figures.

Only scripts that were actually used in the reported workflow should be added. Do not add empty placeholder Python files. The full upstream PyPSA-Earth workflow should be referenced as a dependency rather than copied selectively unless the repository is intended to be independently executable.
