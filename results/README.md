# Results

This folder is intended for small, reviewable result summaries and selected figures. Large solved PyPSA network files are excluded from normal Git tracking and should be distributed through a versioned archive or Git LFS.

## Reported scenario indicators

The current research draft reports three cases:

- **2024 validation:** 46.282 TWh supplied, 1.747 TWh renewable generation and 15.746 MtCO2 direct generator emissions.
- **2030 transition:** 51.033 TWh demand, 21.578 TWh solar and wind output, 11.39% VRE curtailment and 10.075 MtCO2 direct emissions.
- **2050 zero direct emissions:** 109.578 TWh demand, zero direct generator emissions, 21.46% VRE curtailment, and substantial battery, hydrogen and transmission expansion.

These values are scenario outputs, not forecasts. Techno-economic assumptions and annualized cost accounting remain under review, so monetary system-cost comparisons should not be presented as validated results.

Recommended public files include compact CSV summaries for validation, generation, capacity, emissions, curtailment, storage and transmission indicators, together with selected figures generated by the analysis scripts.
