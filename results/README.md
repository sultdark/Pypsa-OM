# Results

This folder contains compact, reviewable outputs from the public PyPSA-Oman scenarios. Large solved PyPSA network files are intentionally excluded from normal Git tracking and may be provided separately on request.

## Files

- `scenario_summary.csv` — consolidated technical indicators for the 2024, 2030 and 2050 scenarios, including demand, generation, capacity, emissions, curtailment, storage and transmission indicators.
- `validation_2024.csv` — comparison of the validated 2024 PyPSA-Oman case with official Nama PWP annual electricity data.
- `figures/` — selected figures used to communicate the model structure and scenario results.

## Interpretation

The reported values are scenario outputs rather than forecasts. The 2024 case is used for base-year validation, the 2030 case is a preliminary transition scenario, and the 2050 case imposes zero direct generator emissions.

Techno-economic assumptions and annualized cost accounting remain under review. Therefore, the repository currently emphasizes non-monetary indicators such as electricity generation, installed capacity, direct CO2 emissions, VRE curtailment, storage deployment and transmission reinforcement.
