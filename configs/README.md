# Scenario configurations

This folder contains the Oman-specific PyPSA-Earth configuration files used for the reported model cases.

Expected files:

- `config_2024_base.yaml` — raw 2024 baseline setup.
- `config_2024_validation.yaml` — calibrated fixed-capacity 2024 validation case.
- `config_2030.yaml` — preliminary 2030 transition scenario.
- `config_2050.yaml` — 2050 zero-direct-generator-emissions scenario.

The scenarios use Oman (`OM`), a clustered electricity network, three-hourly temporal resolution based on the 2013 weather year, and scenario-specific assumptions for generation, storage, conversion and transmission expansion. Monetary inputs remain provisional and should not be interpreted as validated forecasts.
